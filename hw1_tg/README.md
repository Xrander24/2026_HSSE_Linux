# Файловый интерфейс мессенджера

## Предлагаемое решение
2 программы, kernel space - получает запросы на чтение/запись пишет их в fifo. User space этот fifo читает и по запросу формата READ:... WRITE:... выдает данные по чату или записывает в него. В чате хранятся только последние 10 сообщений

## Как запускать
```make``` в папке kernel

```gcc --static tg_server.c -o tg_server``` в папке server

по желанию закинуть это в виртуалку и запускать там

установить модуль через insmod, запустить сервер

Чаты лежат /dev/telegram/chat_<номер чата>
